在使用浏览器搜索时，我们总是会被一些辣鸡网站的信息干扰，因此在搜索时总是需要手动添加 -xxdn, -某乎, -某某文库等字样，这非常麻烦

为了解决这一问题，我开发了chrome插件——`Rubbish-Killer`，用户只需要配置好需要过滤的信息，插件会在查询时自动添加需要过滤的网站信息
